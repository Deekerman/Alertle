How long to reuse a fetched EPG before re-downloading. 0 = no cache.

ESPN Live Verification

Cross-reference ESPN's free API to skip replays — only notify for live or upcoming broadcasts

Notify for replays

Send a notification anyway but prefix the title with [REPLAY] — requires ESPN Live Verification to be enabled above.

When enabled, each matched event is looked up on ESPN before a notification fires:

  • Game not yet started or in progress → notify
  • Game completed > 4 h before broadcast → skip (replay)
  • ESPN has no data for this sport → notify anyway

Covers: NFL, NBA, MLB, NHL, MLS, EPL, La Liga, Bundesliga, Serie A, Ligue 1, UCL, College Football/Basketball.

Description Deduplication

Skip notifications for events whose description text exactly matches one already sent — catches replays without needing ESPN

Works best when your EPG provider uses identical description text for replays. Events with no description are unaffected.
